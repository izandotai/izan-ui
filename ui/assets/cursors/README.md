# Cursor pack drop-in

Drop a cursor pack's `.cur`/`.ani` files here and the shell takes over
cursor shapes at startup (absent files keep the system cursors). File
names follow the classic pack convention:

    Arrow.cur Text.cur Move.cur VResize.cur HResize.cur
    D1Resize.cur D2Resize.cur Link.cur Unavailable.cur

Packs are **not** distributed with izan — most cursor packs (KUDA
included) are licensed for personal use only. The official site links
to the author's download; install it yourself and copy the files in.
